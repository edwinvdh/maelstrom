###
Dockerized setup
###

To start up the containers run the following command from the command line:
```
docker-compose up
```

###
Endpoints
###
The following endpoints are available:
Web interface:
```
localhost:8000
```

Phpmyadmin (usually I using this for DEV purposes):
```
localhost:8082
```
